Outbound link detection

The JavaScript on this page tries to detect all outbound links and set their target attribute to the value _blank. Only outbound links should be affected (and colored green).

Same domain links

  • https://s.codepen.io
  • https://s.codepen.io/login
  • http://user@s.codepen.io
  • http://user:pass@s.codepen.io
  • http://user:pass@s.codepen.io:80
  • //s.codepen.io
  • ftp://s.codepen.io

Codepen seems to run pens under the domain s.codepen.io, so this is to be considered the current domain for any "same domain" check.

Outbound links

  • https://github.com
  • https://github.com/login
  • http://user@github.com
  • http://user:pass@github.com
  • http://user:pass@github.com:80
  • //github.com
  • ftp://github.com
